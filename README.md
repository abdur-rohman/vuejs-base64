# vue-base64 and node-base64

## Project setup
```
npm install
```

## Environment variables (.env file format)
```
PORT=YOUR_PORT
MONGO_URI=YOUR_MONGO_URI_DATABASE
```

### Compiles and hot-reloads for development
```
npm run serve
npm run dev
```

### Compiles and minifies for production (vue)
```
npm run build
```
